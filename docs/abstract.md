# Abstract

We describe a straightedge-and-compass construction producing a constructible approximation \(D(\theta)\) to one third of a given angle \(\theta\). The seed approximation satisfies

```tex
D(\theta)-\frac{\theta}{3}
=
\frac{7}{10368}\theta^3+O(\theta^5),
```

where angles in asymptotic formulae are measured in radians. For a signed approximation error \(e\), the associated residual operator is

```tex
\mathscr{R}(e)
=
e-2\arcsin\!\left(\frac43\sin\frac{3e}{8}\right),
```

with local expansion

```tex
\mathscr{R}(e)
=
-\frac{7}{384}e^3+O(e^5).
```

Repeated correction therefore raises the local error order cubically. Starting from the seed error \(O(\theta^3)\), the first and second corrected angles have errors \(O(\theta^9)\) and \(O(\theta^{27})\). The trigonometric notation is an analytic representation of Euclidean operations on constructed angles and lengths. The method produces arbitrarily refinable finite approximants within the proved local regime; it does not claim exact straightedge-and-compass trisection of an arbitrary angle.
