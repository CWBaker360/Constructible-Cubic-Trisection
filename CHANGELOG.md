# Changelog

All notable repository-level changes are recorded here.

## 1.0.0 - 2026-07-18

Initial research-source release of *A Constructible Cubic Error Cascade for Approximate Angle Trisection*.

Included:

- final LaTeX manuscript and compiled PDF;
- exact seed and residual series derivations;
- finite-stage cubic error cascade through orders \(3\), \(9\), and \(27\);
- standalone symbolic and high-precision numerical verification scripts;
- committed reference outputs;
- repository infographic;
- machine-readable citation metadata;
- reproducibility and publication documentation.

## Publication-preparation update - 2026-07-18

- synchronized README, citation metadata, paper metadata, and repository layout;
- added explicit in-text citations to Wantzel, Dudley, and Rostamian;
- corrected the Rostamian reference title and access information;
- added standalone verification scripts and generated outputs;
- added the final repository infographic;
- added GitHub release notes for version 1.0.0.
- preserved the established dual-license structure: CC BY 4.0 for research materials and MIT for source code.
